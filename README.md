# VisionBased_DrawingSystem
Sistema de desenho em tempo real baseado em visão computacional que utiliza rastreamento de mãos para permitir interação gestual sem contato físico. Desenvolvido com Python, OpenCV e MediaPipe, o projeto interpreta gestos para desenhar, selecionar cores e apagar, aplicando conceitos de processamento de imagem e Human–Computer Interaction (HCI).
